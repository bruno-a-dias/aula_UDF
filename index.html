O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, essencial para desenvolvimento web. Ela permite criar páginas interativas, dinâmicas e modernas, rodando diretamente no navegador do usuário.

Como JavaScript funciona na Web?

  • HTML: Estrutura a página
  • CSS: Estiliza e define o visual
  • JavaScript: Adiciona interatividade e lógica

Exemplo: Ao clicar em um botão, um texto pode ser alterado ou uma animação pode acontecer — tudo isso é feito com JavaScript.

Primeiro Exemplo: Olá, mundo!

<script>
document.querySelector("h1").textContent = "Olá, mundo!";
</script>

Esse é o exemplo clássico para começar em qualquer linguagem de programação. Ele mostra como alterar o conteúdo de um elemento HTML usando JavaScript.

No Console do browser, podemos testar alguns comandos:

console.log("Olá, Mundo!");

alert("Olá, Mundo!");

document.write("Olá, Mundo!");

Variáveis e Tipos de Dados

  • Variáveis: Espaços para guardar valores (números, textos, etc.)
  • Tipos básicos: number, string, boolean, array, object
  • Exemplo:
let nome = "Maria";
let idade = 25;
let aprovado = true;
let notas = [8, 7, 9];
let aluno = { nome: "Maria", idade: 25 };

JavaScript é uma linguagem de tipagem dinâmica: você não precisa declarar o tipo da variável, ele é definido automaticamente.

Estruturas de Controle

Permitem tomar decisões e repetir ações:

  • if/else: Executa um bloco de código se uma condição for verdadeira
  • for/while: Repete um bloco de código várias vezes
if (idade >= 18) {
  console.log("Maior de idade");
} else {
  console.log("Menor de idade");
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Essas estruturas são essenciais para lógica de programação e automação de tarefas.

Funções

Funções agrupam instruções para serem reutilizadas. Você pode passar valores (parâmetros) e receber resultados (retorno). 

function somar(a, b) {
  return a + b;
}
let resultado = somar(3, 5); // resultado = 8

Funções tornam o código mais organizado, facilitando manutenção e reutilização.

Manipulação do DOM

Permite acessar e modificar elementos HTML usando JavaScript:

document.getElementById("meuBotao").innerText = "Clique aqui!";

O DOM (Document Object Model) representa a estrutura da página. Com JavaScript, você pode alterar textos, estilos, adicionar ou remover elementos e responder a eventos do usuário.

Exemplo: Interação com o DOM

<button id="btn">Clique</button>
<p id="msg"></p>
<script>
  document.getElementById("btn").onclick = function() {
    document.getElementById("msg").innerText = "Você clicou!";
  }
</script>

Ao clicar no botão, o texto do parágrafo é alterado. Esse é um exemplo clássico de interatividade com JavaScript.


Armazenamento Local

JavaScript pode guardar informações no navegador:

  • localStorage: Guarda dados mesmo após fechar o navegador
  • cookies: Pequenos dados de sessão
localStorage.setItem("usuario", "Bruno");
let nome = localStorage.getItem("usuario");
console.log("Bem-vindo, " + nome);

Isso é útil para salvar preferências do usuário ou dados temporários.

Mini Projeto: Saudação Personalizada

<input id="nome" placeholder="Seu nome">
<button onclick="salvar()">Salvar</button>
<p id="saudacao"></p>

<script>
function salvar() {
  let nome = document.getElementById("nome").value;
  localStorage.setItem("nomeUsuario", nome);
  document.getElementById("saudacao").innerText = "Olá, " + nome + "!";
}
</script>

Experimente: digite seu nome, clique em "Salvar" e veja a saudação personalizada. O nome fica salvo no navegador!

Dicas para Aprender JavaScript

Exercícios Sugeridos

  • Crie uma função que exiba uma mensagem de boas-vindas ao carregar a página
  • Desenvolva uma função que verifique se um número é par ou ímpar
  • Calcule a média de um array de números
  • Converta uma string para caixa alta
  • Conte a quantidade de vogais em uma palavra
  • Faça um contador de cliques em um botão
  • Crie um mini jogo de adivinhação de número

Praticar com desafios simples é uma das melhores formas de consolidar seu aprendizado em JavaScript.
